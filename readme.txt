How to run this program?
1. Clone this program from https://github.com/vvlnote/cashRegister.git
2. in the local machine, node.js needed to be installed.
3. open the terminal, and go to where this program is located in the local machine
4. type 
		node cashRegister.js
	to run the program.
5. All the test cases are part of program, so you do not need to provide any input.
