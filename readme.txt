How to run this program?
1. Clone this program from https://github.com/vvlnote/telNumberValidator.git
2. in the local machine, node.js needed to be installed.
3. open the terminal, and go to where this program is located in the local machine
4. type 
		node telNumberValiator.js
	to run the program.
5. All the test cases are provided, you do not need to provide any input.
